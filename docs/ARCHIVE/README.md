# Archived Documentation

This folder contains diagnostic, troubleshooting, and temporary fix documentation that has been resolved and archived for historical reference.

These files are kept for:
- Understanding the evolution of the system
- Reference in case similar issues arise
- Audit trail of fixes applied

## Organization

- **Diagnostic Files**: SQL queries used to investigate issues
- **Fix Documentation**: Step-by-step fixes that have been permanently resolved
- **Temporary Migrations**: One-time SQL fixes for specific data issues

## Note
The fixes documented here have been permanently resolved in the codebase. Refer to the main `/docs` folder for current operational documentation.
